# IPL Data Analysis Dashboard 🏏

## Project Overview
The IPL Data Analysis Dashboard is an Excel-based data analytics project that provides insights into Indian Premier League (IPL) seasons from 2008 to 2025. The dashboard helps users explore team performance, championship statistics, and season-wise trends through interactive visualizations.

## Objectives
- Analyze IPL data from 2008–2025.
- Identify winning and runner-up teams.
- Compare team performances across seasons.
- Create interactive dashboards for better decision-making.
- Visualize cricket statistics using charts and pivot tables.
